# Contributing

When contributing to this repository, please first discuss the change you wish to make via issue, email, or any other method with the owners of this repository before making a change.

## Pull Request Process

1. Create a fork of the repository.
2. Setup the bot on your machine to test any changes.
3. Ensure any install or build dependencies are removed before making the PR.
4. If you introduced a new command or a new use case for an existing command, document it at the top of the file.
5. Update the README.md if necessary with details of changes to the interface; this includes file locations and parameters.
6. Format the code with `npx prettier --write .` to prevent merge conflicts and maintain a consistent code style.

## Reaching out for help

Contact any current maintainer through discord, comment on issue, email, or other appropriate channel. Reach out if you need help...

- Setting up the bot on your machine to run inside a test server;
- With an idea that you want to implement but don't know where to start;
- If you get stuck or have minimal coding experience;
- Setting up the CAT managed VPN.
