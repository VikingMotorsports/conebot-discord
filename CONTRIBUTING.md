# Contributing

When contributing to this repository, please first discuss the change you wish to make via issue, email, or any other method with the owners of this repository before making a change.

## Pull Request Process

1. Ensure any install or build dependencies are removed.
2. If you introduced a new command or a new use case for an existing command, document it at the top of the file.
3. Update the README.md if necessary with details of changes to the interface; this file locations and parameters.
4. Format the code with `npx prettier --write .` to prevent merge conflicts and maintain a consistent code style.

## Code of Conduct

Contributions must adhere to the Portland State University code of conduct:

https://www.pdx.edu/dean-student-life/psu-code-student-conduct-and-responsibility
